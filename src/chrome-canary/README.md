How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/chrome-canary`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/chrome-canary`

* Use [`cdnjs`][cdnjs].
