How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/netscape_9`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/netscape_9`

* Use [`cdnjs`][cdnjs].
