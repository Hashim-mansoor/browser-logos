How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/epiphany`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/epiphany`

* Use [`cdnjs`][cdnjs].
