How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/konqueror_4`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/konqueror_4`

* Use [`cdnjs`][cdnjs].
