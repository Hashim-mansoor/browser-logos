How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/internet-explorer-developer-channel`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/internet-explorer-developer-channel`

* Use [`cdnjs`][cdnjs].
