How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/iceweasel`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/iceweasel`

* Use [`cdnjs`][cdnjs].
