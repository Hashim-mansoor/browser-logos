How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/rekonq`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/rekonq`

* Use [`cdnjs`][cdnjs].
