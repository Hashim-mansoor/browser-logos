How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/falkon`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/falkon`

* Use [`cdnjs`][cdnjs].
