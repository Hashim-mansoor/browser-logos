How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/epiphany-technology-preview`
  * [`yarn`][yarn]: `yarn add --dev @browser-logos/epiphany-technology-preview`

* Use [`cdnjs`][cdnjs].
