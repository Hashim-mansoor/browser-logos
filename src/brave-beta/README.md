How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/brave-beta`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/brave-beta`

* Use [`cdnjs`][cdnjs].
