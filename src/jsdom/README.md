How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/jsdom`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/jsdom`

* Use [`cdnjs`][cdnjs].
